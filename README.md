# Customer Purchase Analysis Dashboard using Microsoft Excel

## Overview

This project analyzes customer purchasing behavior using Microsoft Excel. The dataset was cleaned, transformed, and analyzed to identify customer demographics, purchasing trends, and business insights. An interactive dashboard was created using PivotTables, Pivot Charts, Slicers, and Excel functions.

---

## Objectives

- Clean and organize customer sales data
- Analyze purchasing patterns
- Build an interactive Excel dashboard
- Generate business insights
- Support data-driven decision making
